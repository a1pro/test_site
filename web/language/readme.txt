                      MULTI-LANGUAGE SUPPORT
                      
NOTICE. Please don't edit phrases in original language files, because it will be
replaced during upgrade and you will lost all your changes. If you need to edit some
phrases, create file named "en-custom.php" (for English language) in 
amember/language/ folder. You may copy it from sample-custom.php located in 
the same folder and add modified strings into that file.
