
            ALERTPAY INSTALLATION INSTRUCTIONS
1. Enable and configure AlertPay Plugin in aMember control panel.
2. Login into AlertPay Control Panel 
and set "IPN URL" as to {$config.root_url}/plugins/payment/alertpay/ipn.php

Create security code and specify it in plugin configuration

3. Make test purchase. After your testing is done, 
disable testing in aMember Control Panel.
