This plugin is only for demonstration of how credit card plugins are 
working. It only checks a credit card #, cc# 4111-1111-1111-1111 is
accepted, all other cards will be rejected. Please NEVER ENABLE
THIS PLUGIN IN LIVE SYSTEM.
    
                 CCDemo plugin installation

 1. Enable plugin: go to aMember CP -> Setup -> Plugins and enable 
 "cc_demo" payment plugin.
 2. Configure plugin: go to aMember CP -> Setup -> cc_demo 
 and configure it.
 3. Run a test transaction - enter cc# 4111-1111-1111-1111
