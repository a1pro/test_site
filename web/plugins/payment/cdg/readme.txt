    
         CDG plugin installation

 1. Enable plugin: go to aMember CP -> Setup -> Plugins and enable 
 "cdg" payment plugin.
 2. Configure plugin: go to aMember CP -> Setup -> CDG 
 and configure it.
 3. Configure your CDG account - you will need XML transaction enabled
 from your hosting server IP. To find out hosting server IP, please contact
 your hosting provider.
 4. Run a test transaction to ensure everthing is working correctly.
