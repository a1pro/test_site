HOW TO INSTALL
====================================================

1. Copy the centipaid directory into the payment directory
under amember.

2. Enable Centipaid from the payments options in the control
panel as admin.

3. Enter your merchant ID in the centipaid configuration page.

====================================================

Centipaid payment module for amemebers.
Payment plugin based on authoize.net payment module
using Centipaid's CART API version 1.4

For more information about the API please visit
http://www.centipaid.com/api.html
