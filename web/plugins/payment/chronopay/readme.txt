ChronoPay PLUGIN INSTALLATION

1. To use this plugin enable and configure it in aMember control panel.

2. You need to type shared secret value and send it to ChronoPay.

3. You need to ask ChronoPay Support to enable expire notifications.

4. Clients part of ChronoPay system : https://clients.chronopay.com
   For correct work with the site, use Internet Explorer browser.

   Enjoy!
