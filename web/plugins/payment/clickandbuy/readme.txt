
            ClickAndBuy Installation Instructions
1. Enable ClickAndBuy Plugin at aMember CP-> Setup/Configuration -> Plugins
2. Configure plugin at aMember CP-> Setup/Configuration ->  ClickAndBuy
3. The URL for the transaction script should filed in your service area by way of
the transaction link. Please use: {$config.root_surl}/plugins/payment/clickandbuy/ipn.php
4. Make test purchase. After your testing is done, disable testing in aMember CP.

Please note! You should manage recurring payments manually in aMember CP.
