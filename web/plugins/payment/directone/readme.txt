    
           DirectOne plugin installation

 1. Enable plugin: go to aMember CP -> Setup -> Plugins and enable 
 "directone" payment plugin.
 2. Configure plugin: go to aMember CP -> Setup -> DirectOne 
 and configure it, enable test mode.
 3. Run a test transaction to ensure everthing is working correctly. 
  Use the following credit card number for test transaction:
    4242 4242 4242 4242

 4. Disable test mode at aMember CP -> Setup -> DirectOne
