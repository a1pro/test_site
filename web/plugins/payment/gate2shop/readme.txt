
***** IMPORTANT : THIS PLUGIN HAS NOT BEEN TESTED YET ****

1. Enable plugin in aMember CP -> Setup/Configuration -> Plugins

2. Configure it at aMember CP -> Setup/Configuration -> Gate2Shop

3. Configure 'Gate2Shop Currency' at aMember CP -> Manage Products -> Edit

4. Ask Gate2Shop support to configure these URLs for your account:
   Success Page URL: {$config.root_url}/plugins/payment/gate2shop/thanks.php
   Cancel and Failure Page URL: {$config.root_url}/cancel.php

5. Run test transaction to test the integration
