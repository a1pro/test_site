    
                 LogicCommerce plugin installation

 1. Enable plugin: go to aMember CP -> Setup -> Plugins and enable 
 "logiccommerce" payment plugin.
 2. Configure plugin: go to aMember CP -> Setup -> logiccommerce 
 and configure it.
 3. Run a test transaction to ensure everthing is working correctly. 
 While your merchant account is in testing mode, you can use the following
 credit card number : 4242424242424242.
 Use the following test amounts:
    $1.00 processed via the gateway will return "Approved transaction."
    $1.01 processed via the gateway will return "Refer to card issuer"
    $1.62 processed via the gateway will return "Restricted card"
