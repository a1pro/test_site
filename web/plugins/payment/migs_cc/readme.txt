            MIGS payment plugin configuration
           -----------------------------------------

CONFIGURATION
     
1. Enable and configure plugin at "aMember CP -> Setup/Configuration -> Plugins".

2. Configure the plugin at "aMember CP -> Setup/Configuration -> MIGS".

3. Make a test transaction
