           
           NOCHEX plugin installation

 1. Enable APC in your NoChex account:
  * Log in to your NOCHEX account
  * Select the "Edit Automatic Payment Confirmation Details" menu option
  * Edit the URL in the textbox :
     {$config.root_url}/plugins/payment/nochex/ipn.php
  * Press "Save Changes" button
 2. Login to aMember CP, go to Setup/Configuration -> Plugins 
    and enable NoChex plugin. 
 3. Go to aMember CP -> Setup -> NoChex and configure the plugin.
 4. Ask someone to make test purchase.
