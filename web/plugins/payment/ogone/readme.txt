         OGONE payment plugin installation

1. To use this plugin enable and configure it in aMember control panel.

2. In OGONE control panel, go to 
Confuguration -> Technical information, set:

  4.1. PostBack URL to
   {$config.root_url}/plugins/payment/ogone/ipn.php
  4.2 "Make this request in background and differed."
