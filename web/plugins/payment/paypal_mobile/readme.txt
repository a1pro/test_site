                      PayPal Mobile Checkout plugin installation

 1. Enable plugin: go to aMember CP -> Setup/Configuration -> Plugins and enable
	"PayPal Mobile" payment plugin.
 2. Configure plugin: go to aMember CP -> Setup/Configuration -> PayPal Mobile
	and configure it.
 3. Configure PayPal Mobile Currency at aMember CP -> Manage Products -> Edit
 4. Run a test transaction to ensure everything is working correctly.
