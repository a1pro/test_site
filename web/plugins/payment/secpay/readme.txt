    
         secpay plugin installation

 1. Enable plugin: go to aMember CP -> Setup -> Plugins and enable 
 "secpay" payment plugin.
 2. Configure plugin: go to aMember CP -> Setup -> SecPay 
 and configure it.
 3. Run a test transaction to ensure everthing is working correctly. 
 4. Make sure you have enabled and working external cron 
  (aMember CP -> Setup -> Advanced)
