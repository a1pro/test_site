
                       SecurePay Payment Plugin

CONFIGURATION
     
1. Enable and configure plugin at "aMember CP -> Setup -> Plugins".

2. Confiure the plugin at "aMember CP -> Setup -> SecurePay".

3. If you want to enable automatic recurring billing, follow the
     instructions for "Cron Job Configuration."
     http://manual.amember.com/Setup_a_Cron_Job
