         Vanco Services payment plugin installation

 1. Enable plugin: go to aMember CP -> Setup -> Plugins and enable 
    "vanco" payment plugin.
 2. Configure plugin: go to aMember CP -> Setup -> Vanco Services
    and configure it.
    
Set:
    - Vanco company code:
        your company code assigned by Vanco Services
    - Payment Method Title
        displayed on signup page
    - Payment Method Description
        displayed on signup page
        
This plugin doesn't support recurring billing.
This plugin is UNSECURE because Vanco Services does'not propose 
any methods to verify payments.
