
            ZOMBAIO INSTALLATION INSTRUCTIONS
1. Enable Zombaio payment system at aMember CP -> Setup/Configuration -> Plugins.
2. Configure it at aMember CP -> Setup/Configuration -> Zombaio
3. Configure aMember CP -> Manage Products -> Edit -> Zombaio product PRICING ID
4. Set Postback URL (ZScript) for your site at Zombaio merchant account to
   {$config.root_url}/plugins/payment/zombaio/ipn.php
